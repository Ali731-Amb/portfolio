# Portfolio — Alison Amblard

> *"Je suis en train de devenir celle que je cherchais."*

Portfolio personnel développé en HTML, CSS et JavaScript vanilla. Conçu pour présenter mon parcours, mes projets et ma recherche d'alternance en cybersécurité.

🔗 **[Voir le site en ligne](https://ali731-amb.github.io/portfolio)**

---

## À propos

Reconvertie dans le développement web depuis octobre 2025 à [Holberton School](https://www.holbertonschool.fr/), je souhaite me spécialiser en **cybersécurité applicative**.

Avant le code : cuisine gastronomique, travail social, usine, Poste. Ce parcours m'a appris la rigueur, la résistance au stress et l'adaptabilité — des qualités que je retrouve chaque jour dans mon apprentissage.
